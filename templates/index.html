Extração de Relações em Português

Extração de Relações é uma tarefa em Processamento de Linguagem Natural para extrair relações semânticas a partir de texto. Essas relações geralmente ocorrem entre duas ou mais entidades (por exemplo: Pessoa, Organização, Local), o que diretamente associa este estudo ao de Reconhecimento de Entidades Nomeadas.



  • Docker Container Ubuntu 18.04
  • Python 3.6
  • Graphviz v2.40.1
  • Libraries:
    • Flask v1.0.4
    • SpaCy v2.2.3
    • Graphviz v0.13.2
